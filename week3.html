Variables
Variables — Storing Information

A variable is a name that stores a value. In Python you don't need to declare the type — Python figures it out automatically. Think of variables like labelled boxes that hold data.

python — variables

Conditions
If / Elif / Else — Making Decisions

Programs make decisions using if statements. Python uses indentation (4 spaces) to define code blocks — not curly braces like C.

python — conditions
score = int(input("Enter your score (0-100): "))

if score >= 80:
    print(" Excellent! Grade A")
elif score >= 70:
    print("✓ Good job! Grade B")
elif score >= 60:
    print(" Pass. Grade C")
elif score >= 50:
    print(" Borderline. Grade D")
else:
    print(" Below pass mark. Study harder!")
==   Equal to
score == 100 → is score exactly 100?
!=   Not equal
name != "Admin" → is name NOT Admin?
and
Both must be true: age > 18 and is_student == True
or
At least one must be true: day == "Sat" or day == "Sun"
not
Reverses the result: not True → False
For Loop
For Loops — Repeat Automatically

Loops let you repeat code without writing it over and over. The for loop repeats a specific number of times.

python — for loops
# Count 1 to 5
for i in range(1, 6):
    print(f"Count: {i}")

# range(1,6) means 1,2,3,4,5 — stops BEFORE 6

# Loop through a list of names
students = ["Kwame", "Ama", "Kofi", "Abena"]
for student in students:
    print(f"Welcome, {student}!")

# Multiplication table for 3
for i in range(1, 11):
    print(f"3 x {i} = {3 * i}")

While Loop
While Loops — Repeat Until Done

A while loop keeps running as long as a condition is True. Useful when you don't know how many times to repeat.

python — while loop
# Count up to 5
count = 1
while count <= 5:
    print(f"Count is {count}")
    count += 1   # count = count + 1

# Keep asking until correct password
password = ""
while password != "techcamp123":
    password = input("Enter password: ")
    if password != "techcamp123":
        print("Wrong! Try again.")

print("Access granted! ✓")
If your while condition never becomes False, you get an infinite loop — the program never stops. Always make sure something changes inside the loop to eventually break the condition. Press Ctrl+C to stop a running loop.
Lists
Lists — Store Multiple Values

A list stores many items in one variable. Items are separated by commas and surrounded by square brackets [ ].

python — lists
fruits = ["mango", "orange", "banana", "pineapple"]

# Access by index — starts at 0!
print(fruits[0])   # mango
print(fruits[2])   # banana
print(fruits[-1])  # pineapple (last item)

# Useful list methods
fruits.append("watermelon")  # add to end
fruits.remove("orange")      # remove item
fruits.sort()                 # sort alphabetically

print(len(fruits))  # number of items

# Loop through all items
for fruit in fruits:
    print(f" {fruit}")
Lists are zero-indexed — the first item is at index 0, not 1. So a list of 4 items has indexes 0, 1, 2, 3. Index 4 would cause an error!

Number Guessing Game
A classic beginner project. The program picks a random number, you guess it. Uses: random, while loop, if/else, and input. Great for practising all of and 2!

import random

print(" Welcome to the Number Guessing Game!")
print("I'm thinking of a number between 1 and 100.")

attempts = 0

while not guessed:

        if attempts <= 5:
            print(" Amazing! You're a natural guesser.")
        elif attempts <= 10:
            print(" Good effort!")
        else:
            print(" Keep practising your guessing!")

Functions
Functions — Write Once, Use Anywhere

A function is a reusable block of code with a name. Instead of writing the same code 10 times, write it once as a function and call it whenever you need it. Functions make code cleaner, shorter and easier to fix.

python — functions
# Define a function with 'def'
def greet(name):
    print(f"Hello, {name}! Welcome to TechCamp ")

# Call the function
greet("Kwame")
greet("Ama")
greet("Kofi")

# Function that RETURNS a value
def add(a, b):
    return a + b

result = add(10, 25)
print(f"10 + 25 = {result}")   # 35

# Function with default value
def power(base, exponent=2):
    return base ** exponent

print(power(5))      # 25  (5 squared by default)
print(power(2, 10))  # 1024 (2 to the power of 10)
def greet(name):
Defines a function called "greet" that takes one input called "name".
return
Sends a value back to wherever the function was called from.
exponent=2
Default parameter — if you don't pass exponent, it uses 2 automatically.

File Handling
Reading & Writing Files

Real programs save data so it's not lost when the program closes. Python makes file handling very easy with the built-in open() function.

python — file handling
# WRITE — create or overwrite a file
with open("students.txt", "w") as f:
    f.write("Kwame\n")
    f.write("Ama\n")
    f.write("Kofi\n")

# READ — open and read the whole file
with open("students.txt", "r") as f:
    content = f.read()
    print(content)

# APPEND — add to end without deleting
with open("students.txt", "a") as f:
    f.write("Abena\n")

# READ LINES — as a list
with open("students.txt", "r") as f:
    lines = f.readlines()
    for i, name in enumerate(lines, 1):
        print(f"{i}. {name.strip()}")
"w" — Write
Creates file if it doesn't exist. Overwrites if it does.
"r" — Read
Opens file for reading. Error if file doesn't exist.
"a" — Append
Adds to the end. Never deletes existing content.
with...as
Automatically closes the file when done. Always use this!
Error Handling
Try / Except — Handling Errors Gracefully

Programs crash when errors happen. try/except lets you catch errors and handle them without crashing — like a safety net for your code.

python — try/except
# Without try/except — this CRASHES if user types text
# age = int(input("Age: "))  ← ValueError!

# With try/except — handles the error safely
try:
    age = int(input("Enter your age: "))
    print(f"You are {age} years old.")
except ValueError:
    print(" Please enter a number, not text!")

# File reading with error handling
try:
    with open("data.txt", "r") as f:
        print(f.read())
except FileNotFoundError:
    print(" File not found! Check the filename.")
finally:
    print("Done — this runs no matter what.")
try = attempt the risky code. except = what to do if it fails. finally = always runs regardless. This makes your programs professional and unbreakable.

# ── TechCamp Student Registration System ──

def register(name):
    with open("registered.txt", "a") as f:
        f.write(name + "\n")
    print(f"✓ {name} registered successfully!")

def view_all():
    print("\n── Registered Students ──")
    try:
        with open("registered.txt", "r") as f:
            students = f.readlines()
        if not students:
            print("No students registered yet.")
        else:
            for i, s in enumerate(students, 1):
                print(f"  {i}. {s.strip()}")
    except FileNotFoundError:
        print("No students registered yet.")

def count():
    try:
        with open("registered.txt", "r") as f:
            return len(f.readlines())
    except: return 0

print(" TechCamp Registration System")

while True:
    print(f"\nStudents: {count()}")
    print("1. Register  2. View All  3. Quit")
    choice = input("Choice: ").strip()

    if choice == "1":
        name = input("Student name: ").strip()
        if name: register(name)
        else: print("Name cannot be empty.")
    elif choice == "2":
        view_all()
    elif choice == "3":
        print("Goodbye! "); break
Project 2
Password Strength Checker
Combines our hacking knowledge with Python — a tool that checks how strong a password is and gives specific feedback. Great demo of functions, loops, and string methods.
python — password_check.py
def check_password(pw):
    score = 0
    tips  = []

    if len(pw) >= 8:
        score += 1
    else:
        tips.append(" Use at least 8 characters")

    if any(c.isupper() for c in pw):
        score += 1
    else:
        tips.append(" Add uppercase letters (A-Z)")

    if any(c.islower() for c in pw):
        score += 1
    else:
        tips.append(" Add lowercase letters (a-z)")

    if any(c.isdigit() for c in pw):
        score += 1
    else:
        tips.append(" Add numbers (0-9)")

    if any(c in "!@#$%^&*_-" for c in pw):
        score += 1
    else:
        tips.append(" Add symbols (!@#$%^&*)")

    labels = ["", "Very Weak", "Weak", "Fair", "Strong", "Very Strong"]
    emojis = ["", "", "", "", "✓", ""]

    print(f"\nStrength: {emojis[score]} {labels[score]} ({score}/5)")
    if tips:
        print("Improve it:")
        for t in tips: print(f"  {t}")
    else:
        print("Perfect password! ")

while True:
    pw = input("\nEnter password to check (or 'quit'): ")
    if pw == "quit": break
    check_password(pw)

from datetime import datetime
import random

questions = [
    {"q": "What tool in Kali Linux scans for open ports?",
     "a": "nmap", "topic": "Ethical Hacking"},
    {"q": "What does VCC stand for on a circuit?",
     "a": "voltage common collector", "topic": "Arduino"},
    {"q": "What function prints text in Python?",
     "a": "print", "topic": "Python"},
    {"q": "What Arduino function sets a pin HIGH or LOW?",
     "a": "digitalwrite", "topic": "Arduino"},
    {"q": "What command in Kali shows your IP address?",
     "a": "ifconfig", "topic": "Ethical Hacking"},
    {"q": "What keyword defines a function in Python?",
     "a": "def", "topic": "Python"},
]

def run_quiz(player_name):
    score = 0
    random.shuffle(questions)

    print(f"\n Starting quiz for {player_name}!\n")

    for i, q in enumerate(questions, 1):
        print(f"Q{i} [{q['topic']}]: {q['q']}")
        answer = input("   Your answer: ").strip().lower()

        if answer == q["a"].lower():
            print("   ✓ Correct!\n")
            score += 1
        else:
            print(f"    Wrong. Answer: {q['a']}\n")

    percent = (score / len(questions)) * 100
    print(f"── Final Score: {score}/{len(questions)} ({percent:.0f}%) ──")

    if   percent == 100: print(" Perfect! You're a TechCamp legend!")
    elif percent >= 70:  print("✓ Great work! You've learned a lot.")
    elif percent >= 50:  print(" Good effort. Keep studying!")
    else:                print(" Review the notes and try again!")

    # Save result to file
    with open("quiz_results.txt", "a") as f:
        time = datetime.now().strftime("%d/%m/%Y %H:%M")
        f.write(f"{time} | {player_name} | {score}/{len(questions)} | {percent:.0f}%\n")
    print(" Result saved to quiz_results.txt")

# ── Main program ──
print(" TechCamp Boot Camp Quiz")
name = input("Enter your name: ").strip()
run_quiz(name)

again = input("\nPlay again? (yes/no): ")
if again.lower() == "yes":
    run_quiz(name)
else:
    print("Thanks for playing! Keep coding ")

Tips & Limitations
✓ What Works
print(), variables, loops, if/else, functions, lists, dicts, math, random, datetime, string methods — all fully supported.
input() Not Supported
Browser can't pause for keyboard input. Replace input() with a hardcoded value for testing. e.g. name = "Kwame" instead of name = input("Name: ")
